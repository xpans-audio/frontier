# Frontier
A standalone desktop app for rendering spatial audio projects in the xpans 
Ecosystem

## Warning
Frontier is early in development and lacks several features. Some features may
even be broken.

## Building
You must have rustup, Cargo, and npm installed to build Frontier.

Install dependencies:

```sh
$ npm install
```

Build Frontier:

```sh
$ npm run tauri build
```

After compilation, Tauri will generate a bundle for your host operating system.
